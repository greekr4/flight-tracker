# Flight Tracker — 3D 비행 경로 트래커

3D 지구본 위에서 비행 경로를 시각화하고 영상으로 내보낼 수 있는 모바일 웹 앱입니다.

## 주요 기능

- **3D 지구본**: 다크 테마 지구본 + 우주 배경 (globe.gl)
- **경로 순차 입력**: 공항 코드/도시명으로 검색 → 순서대로 추가하면 자동 연결
- **3D 비행기 애니메이션**: Three.js 메쉬가 arc 경로를 따라 순차 이동
- **카메라 추적**: 비행기를 부드럽게 따라가는 카메라
- **재생 컨트롤**: 재생/일시정지, 0.5x/1x/2x 속도 조절
- **영상 내보내기**: WebM 영상 녹화 & 다운로드
- **경로 저장**: localStorage에 자동 저장, 새로고침해도 유지

## 기술 스택

| 영역 | 기술 |
|------|------|
| 프레임워크 | Next.js 15 + React 19 |
| 3D 지구본 | globe.gl + Three.js |
| 스타일링 | Tailwind CSS v4 |
| 영상 내보내기 | MediaRecorder API |
| 배포 | Vercel |

## 사용법

1. 하단 입력창에 공항 검색 (예: `ICN`, `Tokyo`, `JFK`)
2. 선택하면 자동으로 이전 공항과 연결
3. ▶ 버튼으로 애니메이션 재생
4. ⏺ 버튼으로 영상 녹화 & 다운로드

## 로컬 실행

```bash
npm install
npm run dev
```

http://localhost:3000 에서 확인

## 배포

Vercel에 자동 배포됩니다. `main` 브랜치에 푸시하면 자동 빌드 & 배포.
